### Вариант №6. Сервис Специальностей.
#### Добавить Специальность.

Информация требуемая для создания специальности
| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| code | Шифр специальностей | Обязательно | Строка | вида `NN.NN.NN` (цифры, разделённые точками) |  |
| name | Название специальности | Обязательно | Строка |  |  |
| duration_training | Срок обучения в месяцах | Обязательно | Целое число | больше 0 |  |
| id_department | id отделения | Не обязательно | Целое число | больше 0 |  |
| is_active | Статус | Обязательно | Булевое начение | 0(False) или 1(True) | 1(True) |

Информация возвращаемая в случае удачного создания специальности
| Параметр | Тип |
|---|---|
| code | Строка | 
| name | Строка |
| duration_training | Целое число |
| id_department | Целое число |
| is_active | Булевое начение |

#### Изменить специальность по Шифру специальностей

Информация требуемая для изменения специальности
| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| name | Название специальности | Не обязательно | Строка |  |  |
| duration_training | Срок обучения в месяцах | Не обязательно | Целое число | больше 0 |  |
| id_department | id отделения | Не обязательно | Целое число | больше 0 |  |

Информация возвращаемая в случае удачного изменения специальности
| Параметр | Тип |
|---|---|
| code | Строка | 
| name | Строка |
| duration_training | Целое число |
| id_department | Целое число |
| is_active | Булевое начение |

#### Удалить специальность по Шифру специальностей

Вернет, если специальность была закрыта(удалена)
| Параметр | Тип | Сообщение |
|---|---|---|
| message | Строка | Спецаильность удалена(закрыта) |

Вернет ошибку, если специальность не закрыта(удалена)
| Параметр | Тип | Сообщение |
|---|---|---|
| message | Строка | Спецаильность не найдена или не удалось закрыть(удалить) |

Информация возвращаемая в случае удачного поиска специальности

#### Получить список специальностей по заданным параметрам

Информация требуемая для получения списка специальностей
| Параметр | Тип | Описание |
|---|---|---|
| name | Строка | Поиск будет частичный, не учитывая регистр |
| duration_training | Целое число | "1" - равно 1 |
| min_duration_training | Целое число | "1" - от 1 |
| max_duration_training | Целое число | "10" - до 10 |
| id_department | Целое число | "1" - равно 1 |
| min_id_department | Целое число | "1" - от 1 |
| max_id_department | Целое число | "10" - до 10 |
| is_active | Булевое начение | 0(False) или 1(True) |

Информация возвращается в виде списка специальностей
| Параметр | Тип |
|---|---|
| code | Строка | 
| name | Строка |
| duration_training | Целое число |
| id_department | Целое число |
| is_active | Булевое начение |
