### Вариант №3. Role Service (Сервис ролей).
#### Добавить роль.

Информация, требуемая для создания роли:
| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| name | Название | Обязательно | Строка | Уникальное значение | — |
| access_level | Уровень доступа | Обязательно | Целое число | Больше 0 (от 1 до n) | 1 |
| description | Описание | Не обязательно | Строка | — | `NULL` |

Информация, возращаемая в случае удачного создания роли:
| Параметр | Тип |
|---|---|
| id | Целое число |
| name | Строка |
| access_level | Целое число |
| description | Строка |
| is_active | Булево значение |

#### Измененить роль по ID.

Информация, требуемая для изменения роли по ID:
| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| name | Название | Обязательно | Строка | Уникальное значение | — |
| access_level | Уровень доступа | Обязательно | Целое число | Больше 0 (от 1 до n) | 1 |
| description | Описание | Не обязательно | Строка | — | `NULL` |

Информация, возращаемая в случае удачного изменения роли:
| Параметр | Тип |
|---|---|
| id | Целое число |
| name | Строка |
| access_level | Целое число |
| description | Строка |
| is_active | Булево значение |

#### Удаление роли по ID.

Если роль активна (не удалена), вернёт:
| Параметр | Тип | Сообщение |
|---|---|---|
|message| Строка| Роль не удалена или не найдена |

Иначе, если роль закрыта (удалена), вернёт:
| Параметр | Тип | Сообщение |
|---|---|---|
|message| Строка| Роль удалена |


#### Получить роль по ID.

Информация, возвращаемая в случае удачного поиска роли по ID:
| Параметр | Пояснение | Тип | 
|---|---|---|
| id | Идентификатор | Целое число |
| name | Название | Строка |
| access_level | Уровень доступа | Целое число |
| description | Описание | Строка |
| is_active | Статус |Булево значение |

#### Получить список ролей по заданным параметрам.

Информация, требуемая для получения списка ролей по заданным параметрам:
| Параметр | Пояснение | Тип | Описание |
|---|---|---|---|
| access_level_equally | Точное число "n" | Целое число | ровно "1" |
| access_level_greater | Перечисление "больше n" | Целое число | больше "1" |
| access_level_less | Перечисление "меньше n" | Целое число | меньше "6" |

Информация, возвращаемая в виде списка ролей:
| Параметр | Тип |
|---|---|
| id | Целое число |
| name | Строка |
| is_active | Булево значение |
