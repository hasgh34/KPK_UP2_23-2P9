# Вариант №1. Сервис аутентификации.

### 1. Добавить пользователя (регистрация)  
Информация требуемая для создания пользователя

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
| ----- | ----- | ----- | ----- | ----- | ----- |
| login | Логин | Обязательно | Строка | Уникальный, от 3 до 50 символов, только латиница, цифры и _ |  |
| password | Пароль | Обязательно | Строка | Не менее 8 символов |  |

**Выходные данные**

Информация возвращаемая в случае удачного создания пользователя

| Параметр | Пояснение | Тип |
| :---- | :---- | :---- |
| id | id пользователя | Целое число |
| login | Логин | Строка |
| is_active | Статус | Булево значение |

### 2. Вход в учетную запись  
**Информация требуемая для входа в систему**

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :---- | :---- | :---- | :---- | :---- | :---- |
| login | Логин | Обязательно | Строка |  |  |
| password | Пароль | Обязательно | Строка |  |  |

**Выходные данные**  
Информация, возвращаемая в случае удачного входа

| Параметр | Пояснение | Тип |
| :---- | :---- | :---- |
| access_token | Токен доступа | Строка (JWT) |
| refresh_token | Токен обновления | Строка (JWT) |

access_token дополнительно хэширует role_id

### 3. Изменить пользователя по ID.
Информация, требуемая для изменения пользователя по ID.

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :---- | :---- | :---- | :---- | :---- | :---- |
| login | Логин | Обязательно | Строка | Уникальный | NULL |

**Выходные данные**  
Информация, требуемая для изменения пользователя по ID.

| Параметр | Пояснение | Тип |
| :---- | :---- | :---- |
| id | id пользователя | Целое число |
| login | Логин | Строка |
| is_active | Статус | Булево значение |

### 4. Сброс пароля.

| Параметр | Пояснение | Обязательность | Тип | Ограничение |
| :---- | :---- | :---- | :---- | :---- |
| login | Логин | Обязательно | Строка | Должен существовать в системе |

**Выходные данные**

| Параметр | Тип | Сообщение |
| :---- | :---- | :---- |
| message | Строка | Письмо отправлено на email |

| Параметр | Тип | Сообщение |
| :---- | :---- | :---- |
| message | Строка | Письмо не отправлено на email |

Информация требуемая для подтверждения сброса пароля

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :---- | ----- | :---- | :---- | :---- | :---- |
| reset_token | Токен сброса | Обязательно | Строка | Валидный токен из ссылки в письме |  |
| new_password | Новый пароль | Обязательно | Строка | Не менее 8 символов, не совпадает со старым |  |

**Выходные данные**  
Вернет, если пароль успешно изменен.

| Параметр | Тип | Сообщение |
| :---- | :---- | :---- |
| message | Строка | Пароль успешно изменен |

Вернет, если пароль не изменен.

| Параметр | Тип | Сообщение |
| :---- | :---- | :---- |
| message | Строка | Пароль не изменен |
